# Django Blog Project

A Django blog web application with user authentication and post interactions.

## Features

- User Registration & Login
- User Profile System
- Create, Update, Delete Posts
- Comments System
- Like System
- CKEditor Integration

## Tech Stack

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

```bash
git clone https://github.com/ajarpitjain2007-afk/django-blog-project.git
cd django-blog-project

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```
